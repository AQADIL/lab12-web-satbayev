# AkadiEdu - Образовательная платформа

## Реализованные компоненты

### 1. Хедер-меню
- Фиксированная навигационная панель с логотипом
- Адаптивное меню для мобильных устройств
- Плавные переходы и hover-эффекты

### 2. Выпадающее меню (Dropdown)
- Pure CSS dropdown при наведении на "Программы"
- Плавная анимация появления
- Подменю с категориями курсов

### 3. Боковое меню (Drawer)
- Кнопка гамбургер для мобильных устройств
- Выдвижное меню справа с transform: translateX
- Backdrop overlay для закрытия по клику вне меню
- Поддержка клавиши Escape для закрытия

### 4. Кнопка «Поделиться»
- Floating Action Button в правом нижнем углу
- Раскрывающееся меню с иконками соцсетей
- Поддержка Facebook, Twitter, LinkedIn, Telegram
- Закрытие по клику вне меню

### 5. Прогресс-бары (Loading bars)
- Анимированное заполнение при загрузке страницы
- Различные проценты завершения для каждого навыка
- Градиентная заливка от primary до secondary цвета
- Плавная анимация 1.5 секунды

### 6. Карточки с эффектом прозрачности
- Сетка из 6 карточек курсов
- Начальная прозрачность opacity: 0.6
- Hover эффект: opacity: 1.0 + scale(1.02)
- Дополнительная анимация изображений при наведении
- Click анимация для интерактивности

## Технические особенности

- **CSS Variables** для управления цветовой схемой
- **Grid и Flexbox** для адаптивной верстки
- **CSS Transitions** для плавных анимаций
- **Semantic HTML5** структура
- **Vanilla JavaScript** без зависимостей
- **Mobile-first** подход с медиа-запросами

## Структура файлов

```
lab11/
├── index.html      # Основная HTML разметка
├── style.css       # Стили и анимации
├── script.js       # Интерактивность
└── README.md       # Документация
```

## Как использовать

1. Откройте `index.html` в браузере
2. Все компоненты полностью функциональны
3. Проверьте адаптивность через инструменты разработчика

## Цветовая схема

- Primary: #4F46E5 (индиго)
- Secondary: #10B981 (зеленый)
- Accent: #F59E0B (желтый)
- Text: #1F2937 (темно-серый)
- Background: #F9FAFB (светло-серый)
