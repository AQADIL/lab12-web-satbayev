О программе

Программа Backend разработки в MarleneEdu научит вас создавать надежные и масштабируемые серверные приложения. Вы освоите современные технологии и подходы к разработке API, работе с базами данных и развертыванию приложений.

Чему вы научитесь:

  • ✓ Разрабатывать RESTful API и GraphQL сервисы
  • ✓ Работать с реляционными и NoSQL базами данных
  • ✓ Использовать Node.js, Python, Java для backend разработки
  • ✓ Применять микросервисную архитектуру
  • ✓ Обеспечивать безопасность приложений
  • ✓ Развертывать приложения в Docker и Kubernetes
